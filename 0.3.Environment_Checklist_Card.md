## 🚀 The smoke test (proves the whole chain works)

```bash
mkdir ~/100x-smoketest && cd ~/100x-smoketest
claude
```
Then type at the Claude prompt:

> **Create a file called hello.txt containing today's date and a one-line hello message.**

Approve it → check it worked:

```bash
cat hello.txt      # Windows PowerShell: type hello.txt
```

**See your message? You're ready. 🎉**

## 🧯 Fast fixes

- **`claude: command not found`** → npm global bin not on PATH. Reopen terminal; ask a TA.
- **Node < 18** → macOS: `brew upgrade node`. WSL: install NodeSource LTS.
- **Login page won't open** → copy the URL from the terminal into your browser manually.
- **Locked-down work laptop** → you need admin rights; use a personal laptop if possible.
